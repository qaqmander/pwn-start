# pwn-start

ctf夺旗赛之pwn，从这里开始。

## 什么是pwn

pwn就是通过和程序进行一些交互，来使程序运行/bin/sh，从而拿到shell

## 工具

1. 反编译工具，如ida
2. 调试工具，如gdb（但一般用gdb的插件，经典的有peda，gef和pwndbg，推荐gef
3. python2的第三方库，pwntools
